Chuleta de Clases Esenciales de Tailwind

Tu referencia rápida para construir interfaces como un profesional.

1. Espaciado (Padding y Margin)

  • p-{tamaño}: Añade padding (relleno interno). Ejemplos: p-4, px-4 (eje X), py-2 (eje Y), pt-6 (top).
  • m-{tamaño}: Añade margin (margen externo). Ejemplos: m-8, mx-auto (para centrar horizontalmente).
  • space-y-{tamaño}: Añade un espacio vertical entre elementos hijos. Muy útil para listas.

2. Dimensiones (Ancho y Alto)

  • w-{tamaño}: Define el width (ancho). Ejemplos: w-full, w-1/2, max-w-2xl (ancho máximo).
  • h-{tamaño}: Define el height (alto). Ejemplos: h-full, h-screen.

3. Layout (Flexbox y Grid)

  • flex: Activa Flexbox. Clases de ayuda: flex-col, items-center, justify-center, justify-between.
  • grid: Activa Grid. Clases de ayuda: grid-cols-3 (3 columnas), gap-4 (espacio entre elementos).

4. Tipografía (Texto)

  • text-{tamaño}: Tamaño de la fuente. Ejemplos: text-sm, text-base, text-lg, text-2xl.
  • font-{grosor}: Grosor de la fuente. Ejemplos: font-medium, font-semibold, font-bold.
  • text-{color}-{intensidad}: Color del texto. Ejemplos: text-gray-500, text-indigo-600.

5. Fondos y Bordes

  • bg-{color}-{intensidad}: Color de fondo. Ejemplos: bg-white, bg-gray-100.
  • border: Añade un borde. Clases de ayuda: border-b (solo abajo), border-gray-300 (color).
  • rounded-{tamaño}: Redondea las esquinas. Ejemplos: rounded-lg, rounded-full.

6. Sombras e Interactividad

  • shadow-{tamaño}: Añade una sombra. Ejemplos: shadow-md, shadow-lg.
  • hover:{clase}: Aplica un estilo al pasar el ratón. Ejemplo: hover:bg-blue-700.
  • cursor-pointer: Muestra el cursor de "manita" al pasar el ratón.
